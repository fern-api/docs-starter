<Indent>

Prefetch all virtual environments that will be used by workers.

**Parameters:**

<ParamField path="filters" default="None">
List of strings to match against actor FQNs. If provided, only
    actors whose FQN contains at least one of the filter strings will
    be prefetched. If None, all venvs are prefetched.
</ParamField>

<ParamField path="negative_filters" default="None">
List of strings to exclude from prefetching. Actors whose
    FQN contains any of these strings will be skipped.
</ParamField>


</Indent>
