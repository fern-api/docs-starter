```cpp showLineNumbers={false}
#include <cuda/event_ref.h>
```

## Constructors

### event_ref <Badge intent="note" minimal>inline</Badge> <Badge intent="note" minimal>constexpr</Badge> <Badge intent="note" minimal>noexcept</Badge>

<Tabs>
<Tab title="From native handle">

Construct a new `event_ref` object from a `cudaEvent_t`.

This constructor provides an implicit conversion from `cudaEvent_t`

<CodeBlock links={{"event_ref": "/libcudacxx/api/cuda::event_ref"}}>
```cpp showLineNumbers={false}
cuda::event_ref::event_ref(
    ::cudaEvent_t __evnt
) noexcept
```
</CodeBlock>

<Note>
: It is the callers responsibility to ensure the `event_ref` does not outlive the event denoted by the `cudaEvent_t` handle.
</Note>

<Info title="Postconditions">
[`get()`](/libcudacxx/api/cuda::event_ref::get())` == __evnt`
</Info>

### is_done <Badge intent="note" minimal>inline</Badge> <Badge intent="note" minimal>const</Badge> <Badge intent="note" minimal>nodiscard</Badge>

If is_done returns true, calling [sync()](/libcudacxx/api/cuda::event_ref::sync()) on this event will return immediately
