`memory_resource` is the base class for all other memory resources.

```cpp showLineNumbers={false}
#include <thrust/mr/memory_resource.h>
```

<AccordionGroup>
<Accordion title="Template parameters">

<ParamField path="Pointer" type="typename" default="void*">
The pointer type that is allocated and deallocated by the memory resource derived from this base class. If this is `void *`, this class derives from `std::pmr::memory_resource`.
</ParamField>

</Accordion>
</AccordionGroup>

### allocate <Badge intent="note" minimal>inline</Badge> <Badge intent="note" minimal>nodiscard</Badge>
