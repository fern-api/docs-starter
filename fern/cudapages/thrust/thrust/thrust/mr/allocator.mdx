An `mr::allocator` is a template that fulfills the C++ requirements for Allocators, allowing to use the NPA-based memory resources where an Allocator is required.

Unlike memory resources, but like other allocators, `mr::allocator` is typed and bound to allocate object of a specific type, however it can be freely rebound to other types.

```cpp showLineNumbers={false}
#include <thrust/mr/allocator.h>
```

<AccordionGroup>
<Accordion title="Template parameters">

<ParamField path="T" type="typename">
The type that will be allocated by this allocator.
</ParamField>

<ParamField path="MR" type="class">
The upstream memory resource to use for memory allocation. Must derive from [`thrust::mr::memory_resource`](/library/api/thrust::mr::memory_resource) and must be `final` (in C++11 and beyond).
</ParamField>

</Accordion>
</AccordionGroup>

**Inherits from:** `thrust::mr::validator< MR >` (private)
