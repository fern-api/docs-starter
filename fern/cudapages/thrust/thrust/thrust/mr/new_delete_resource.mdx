A memory resource that uses global operators new and delete to allocate and deallocate memory.

Uses alignment-enabled overloads when available, otherwise uses regular overloads and implements alignment requirements by itself.

```cpp showLineNumbers={false}
#include <thrust/mr/new.h>
```

**Inherits from:** `thrust::mr::new_delete_resource_base` (public)

This class is marked <Badge intent="note" minimal>final</Badge>.
