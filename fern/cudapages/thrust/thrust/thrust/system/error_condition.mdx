The class `error_condition` describes an object used to hold values identifying error conditions.

```cpp showLineNumbers={false}
#include <thrust/system/error_code.h>
```

<Note>
`error_condition` values are portable abstractions, while [`error_code`](/library/api/thrust::system::error_code) values are implementation specific.
</Note>
