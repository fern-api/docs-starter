`project1st` is a function object that takes two arguments and returns its first argument; the second argument is unused.

It is essentially a generalization of identity to the case of a Binary Function.

```cpp showLineNumbers={false}
#include <thrust/functional.h>
```

**See also:**
identity,
[project2nd](/library/api/thrust::project2nd)

## Example

```cpp showLineNumbers={false}
#include <thrust/functional.h>
#include <assert.h>
...
int x =  137;
int y = -137;
thrust::project1st<int> pj1;
assert(x == pj1(x,y));
```
