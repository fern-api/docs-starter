`project2nd` is a function object that takes two arguments and returns its second argument; the first argument is unused.

It is essentially a generalization of identity to the case of a Binary Function.

```cpp showLineNumbers={false}
#include <thrust/functional.h>
```

**See also:**
identity,
[project1st](/library/api/thrust::project1st)

## Example

```cpp showLineNumbers={false}
#include <thrust/functional.h>
#include <assert.h>
...
int x =  137;
int y = -137;
thrust::project2nd<int> pj2;
assert(y == pj2(x,y));
```
