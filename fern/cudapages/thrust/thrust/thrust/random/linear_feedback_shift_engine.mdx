A `linear_feedback_shift_engine` random number engine produces unsigned integer random values using a linear feedback shift random number generation algorithm.

```cpp showLineNumbers={false}
#include <thrust/random/linear_feedback_shift_engine.h>
```

<Note>
`linear_feedback_shift_engine` is based on the Boost Template Library's linear_feedback_shift.
</Note>

<AccordionGroup>
<Accordion title="Template parameters">

<ParamField path="UIntType" type="typename">
The type of unsigned integer to produce.
</ParamField>

<ParamField path="w" type="size_t">
The word size of the produced values (`w <= sizeof(UIntType)`).
</ParamField>

<ParamField path="k" type="size_t">
The k parameter of Tausworthe's 1965 algorithm.
</ParamField>

<ParamField path="q" type="size_t">
The q exponent of Tausworthe's 1965 algorithm.
</ParamField>

<ParamField path="s" type="size_t">
The step size of Tausworthe's 1965 algorithm.
</ParamField>

</Accordion>
</AccordionGroup>
