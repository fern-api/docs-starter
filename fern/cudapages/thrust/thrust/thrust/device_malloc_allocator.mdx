`device_malloc_allocator` is a device memory allocator that employs the `device_malloc` function for allocation.

`device_malloc_allocator` is deprecated in favor of [`thrust::mr`](/library/api/thrust::mr) memory resource-based allocators.

```cpp showLineNumbers={false}
#include <thrust/device_malloc_allocator.h>
```

**See also:**
device_malloc,
[device_ptr](/library/api/thrust::device_ptr),
[device_allocator](/library/api/thrust::device_allocator),
[https://en.cppreference.com/w/cpp/memory/allocator](https://en.cppreference.com/w/cpp/memory/allocator)
