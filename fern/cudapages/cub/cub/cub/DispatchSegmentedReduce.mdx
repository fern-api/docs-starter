template <typename MaxPolicyT>

**Parameters**

<ParamField path="d_temp_storage" type="void *">
Device-accessible allocation of temporary storage. When `nullptr`, the required allocation size is written to [`temp_storage_bytes`](/library/api/cub::_dispatch_segmented_reduce::temp_storage_bytes) and no work is done.
</ParamField>

<ParamField path="temp_storage_bytes" type="size_t &">
Reference to size in bytes of [`d_temp_storage`](/library/api/cub::_dispatch_segmented_reduce::d_temp_storage) allocation
</ParamField>

<ParamField path="d_in" type="InputIteratorT">
Pointer to the input sequence of data items
</ParamField>

<ParamField path="d_out" type="OutputIteratorT">
Pointer to the output aggregate
</ParamField>

<ParamField path="num_segments" type="::cuda::std::int64_t">
The number of segments that comprise the sorting data
</ParamField>

<ParamField path="d_begin_offsets" type="BeginOffsetIteratorT">
Random-access input iterator to the sequence of beginning offsets of length [`num_segments`](/library/api/cub::_dispatch_segmented_reduce::num_segments), such that [`d_begin_offsets`](/library/api/cub::_dispatch_segmented_reduce::d_begin_offsets)`[i]` is the first element of the *i*<sup>th</sup> data segment in `d_keys_*` and `d_values_*`
</ParamField>

<ParamField path="d_end_offsets" type="EndOffsetIteratorT">
Random-access input iterator to the sequence of ending offsets of length [`num_segments`](/library/api/cub::_dispatch_segmented_reduce::num_segments), such that [`d_end_offsets`](/library/api/cub::_dispatch_segmented_reduce::d_end_offsets)`[i] - 1` is the last element of the *i*<sup>th</sup> data segment in `d_keys_*` and `d_values_*`. If [`d_end_offsets`](/library/api/cub::_dispatch_segmented_reduce::d_end_offsets)`[i] - 1 <= `[`d_begin_offsets`](/library/api/cub::_dispatch_segmented_reduce::d_begin_offsets)`[i]`, the *i*<sup>th</sup> is considered empty.
</ParamField>

<ParamField path="reduction_op" type="ReductionOpT">
Binary reduction functor
</ParamField>

<ParamField path="init" type="InitT">
The initial value of the reduction
</ParamField>

<ParamField path="stream" type="cudaStream_t">
**[optional]** CUDA stream to launch kernels within. Default is stream<sub>0</sub>.
</ParamField>
