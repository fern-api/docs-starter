**Parameters**

<ParamField path="d_temp_storage" type="void *">
Device-accessible allocation of temporary storage. When nullptr, the required allocation size is written to [`temp_storage_bytes`](/library/api/cub::_dispatch_segmented_radix_sort::temp_storage_bytes) and no work is done.
</ParamField>

<ParamField path="temp_storage_bytes" type="size_t &">
Reference to size in bytes of [`d_temp_storage`](/library/api/cub::_dispatch_segmented_radix_sort::d_temp_storage) allocation
</ParamField>

<ParamField path="d_keys" type="DoubleBuffer<KeyT> &">
Double-buffer whose current buffer contains the unsorted input keys and, upon return, is updated to point to the sorted output keys
</ParamField>

<ParamField path="d_values" type="DoubleBuffer<ValueT> &">
Double-buffer whose current buffer contains the unsorted input values and, upon return, is updated to point to the sorted output values
</ParamField>

<ParamField path="num_items" type="::cuda::std::int64_t">
Number of items to sort
</ParamField>

<ParamField path="num_segments" type="::cuda::std::int64_t">
The number of segments that comprise the sorting data
</ParamField>

<ParamField path="d_begin_offsets" type="BeginOffsetIteratorT">
Random-access input iterator to the sequence of beginning offsets of length [`num_segments`](/library/api/cub::_dispatch_segmented_radix_sort::num_segments), such that [`d_begin_offsets`](/library/api/cub::_dispatch_segmented_radix_sort::d_begin_offsets)`[i]` is the first element of the *i*<sup>th</sup> data segment in `d_keys_*` and `d_values_*`
</ParamField>

<ParamField path="d_end_offsets" type="EndOffsetIteratorT">
Random-access input iterator to the sequence of ending offsets of length [`num_segments`](/library/api/cub::_dispatch_segmented_radix_sort::num_segments), such that [`d_end_offsets`](/library/api/cub::_dispatch_segmented_radix_sort::d_end_offsets)`[i]-1` is the last element of the *i*<sup>th</sup> data segment in `d_keys_*` and `d_values_*`. If [`d_end_offsets`](/library/api/cub::_dispatch_segmented_radix_sort::d_end_offsets)`[i]-1` &lt;= [`d_begin_offsets`](/library/api/cub::_dispatch_segmented_radix_sort::d_begin_offsets)`[i]`, the *i*<sup>th</sup> is considered empty.
</ParamField>

<ParamField path="begin_bit" type="int">
The beginning (least-significant) bit index needed for key comparison
</ParamField>

<ParamField path="end_bit" type="int">
The past-the-end (most-significant) bit index needed for key comparison
</ParamField>

<ParamField path="is_overwrite_okay" type="bool">
Whether is okay to overwrite source buffers
</ParamField>

<ParamField path="stream" type="cudaStream_t">
CUDA stream to launch kernels within. Default is stream<sub>0</sub>.
</ParamField>
