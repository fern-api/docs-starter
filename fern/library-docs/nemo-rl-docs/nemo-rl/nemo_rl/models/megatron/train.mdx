<Indent>

Aggregate training statistics across microbatches and data-parallel ranks.

Computes a global loss by all-reducing per-gradient-buffer losses across the
data-parallel group, then collects per-microbatch metrics into lists keyed by
metric name.

**Parameters:**

<ParamField path="all_mb_metrics" type="List[Dict[str, Any]]">
List of metric dicts from each microbatch.
</ParamField>

<ParamField path="losses" type="List[float]">
List of per-gradient-buffer scalar losses on this rank.
</ParamField>

<ParamField path="data_parallel_group" type="torch.distributed.ProcessGroup">
The data-parallel process group for all-reduce.
</ParamField>

**Returns:** `Tuple[Dict[str, List[Any]], torch.Tensor]`

Tuple of:
- mb_metrics: Dict mapping metric names to lists of values across microbatches.
- global_loss: Tensor of losses summed across all data-parallel ranks.


</Indent>
