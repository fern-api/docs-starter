Create a tool to do retrieval of documents.

**Parameters:**

<ParamField path="retriever" type="BaseRetriever">
The retriever to use for the retrieval
</ParamField>

<ParamField path="name" type="str">
The name for the tool.

This will be passed to the language model, so should be unique and somewhat
descriptive.
</ParamField>

<ParamField path="description" type="str">
The description for the tool.

This will be passed to the language model, so should be descriptive.
</ParamField>

<ParamField path="document_prompt" type="BasePromptTemplate | None" default="None">
The prompt to use for the document.
</ParamField>

<ParamField path="document_separator" type="str" default="'\n\n'">
The separator to use between documents.
</ParamField>

<ParamField path="response_format" type="Literal['content', 'content_and_artifact']" default="'content'">
The tool response format.

If `'content'` then the output of the tool is interpreted as the contents of
a `ToolMessage`. If `'content_and_artifact'` then the output is expected to
be a two-tuple corresponding to the `(content, artifact)` of a `ToolMessage`
(artifact being a list of documents in this case).
</ParamField>

**Returns:** `StructuredTool`

Tool class to pass to an agent.


</Indent>
