Retrieve a value from the current scope using a dot-separated key path.

Traverses through nested dictionaries and lists using dot notation.

Supports special key `'.'` to return the current scope.

**Parameters:**

<ParamField path="key" type="str">
Dot-separated key path (e.g., `'user.name'` or `'.'` for current scope).
</ParamField>

<ParamField path="scopes" type="Scopes">
List of scope dictionaries to search through.
</ParamField>

<ParamField path="warn" type="bool">
Whether to log a warning when a key is not found.
</ParamField>

<ParamField path="keep" type="bool">
Whether to return the original template tag when key is not found.
</ParamField>

<ParamField path="def_ldel" type="str">
Left delimiter for template (used when keep is `True`).
</ParamField>

<ParamField path="def_rdel" type="str">
Right delimiter for template (used when keep is `True`).
</ParamField>

**Returns:** `Any`

The value found at the key path.

If not found, returns the original template tag when keep is `True`,
otherwise returns an empty string.


</Indent>

<Indent>

Render a mustache template.

Renders a mustache template with a data scope and inline partial capability.

**Parameters:**

<ParamField path="template" type="str | list[tuple[str, str]]" default="''">
A file-like object or a string containing the template.
</ParamField>

<ParamField path="data" type="Mapping[str, Any]" default="EMPTY_DICT">
A python dictionary with your data scope.
</ParamField>

<ParamField path="partials_dict" type="Mapping[str, str]" default="EMPTY_DICT">
A python dictionary which will be search for partials
before the filesystem is.

`&#123;'include': 'foo'&#125;` is the same as a file called include.mustache
(defaults to `&#123;&#125;`).
</ParamField>

<ParamField path="padding" type="str" default="''">
This is for padding partials, and shouldn't be used
(but can be if you really want to).
</ParamField>

<ParamField path="def_ldel" type="str" default="'&#123;&#123;'">
The default left delimiter

(`'&#123;&#123;'` by default, as in spec compliant mustache).
</ParamField>

<ParamField path="def_rdel" type="str" default="'&#125;&#125;'">
The default right delimiter

(`'&#125;&#125;'` by default, as in spec compliant mustache).
</ParamField>

<ParamField path="scopes" type="Scopes | None" default="None">
The list of scopes that `get_key` will look through.
</ParamField>

<ParamField path="warn" type="bool" default="False">
Log a warning when a template substitution isn't found in the data
</ParamField>

<ParamField path="keep" type="bool" default="False">
Keep unreplaced tags when a substitution isn't found in the data.
</ParamField>

**Returns:** `str`

A string containing the rendered template.


</Indent>

Tokenize a mustache template.

Tokenizes a mustache template in a generator fashion, using file-like objects. It
also accepts a string containing the template.

**Parameters:**

a file-like object, or a string of a mustache template
</ParamField>

(`'&#123;&#123;'` by default, as in spec compliant mustache)
</ParamField>

<ParamField path="def_rdel" type="str" default="'&#125;&#125;'">
The default right delimiter
(`'&#125;&#125;'` by default, as in spec compliant mustache)
</ParamField>
