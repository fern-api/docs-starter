<Indent>

Extract all links from a raw HTML string and convert into absolute paths.

**Parameters:**

<ParamField path="raw_html" type="str">
Original HTML.
</ParamField>

<ParamField path="url" type="str">
The url of the HTML.
</ParamField>

<ParamField path="base_url" type="str | None" default="None">
the base URL to check for outside links against.
</ParamField>

<ParamField path="pattern" type="str | re.Pattern | None" default="None">
Regex to use for extracting links from raw HTML.
</ParamField>

<ParamField path="prevent_outside" type="bool" default="True">
If `True`, ignore external links which are not children
of the base URL.
</ParamField>

<ParamField path="exclude_prefixes" type="Sequence[str]" default="()">
Exclude any URLs that start with one of these prefixes.
</ParamField>

<ParamField path="continue_on_failure" type="bool" default="False">
If `True`, continue if parsing a specific link raises an
exception. Otherwise, raise the exception.
</ParamField>

**Returns:** `list[str]`

A list of absolute paths to sub links.


</Indent>
