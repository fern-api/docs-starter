<Indent>

Row-wise cosine similarity between two equal-width matrices.

**Parameters:**

<ParamField path="x" type="Matrix">
A matrix of shape `(n, m)`.
</ParamField>

<ParamField path="y" type="Matrix">
A matrix of shape `(k, m)`.
</ParamField>

**Returns:** `np.ndarray`

A matrix of shape `(n, k)` where each element `(i, j)` is the cosine similarity
between the `i`th row of `x` and the `j`th row of `y`.

**Raises:**

- `ValueError`: If the number of columns in `x` and `y` are not the same.
- `ImportError`: If numpy is not installed.


</Indent>
