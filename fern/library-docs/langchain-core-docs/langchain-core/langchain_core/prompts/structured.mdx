<Badge>classmethod</Badge>

Create a chat prompt template from a variety of message formats.

**Parameters:**

<ParamField path="messages" type="Sequence[MessageLikeRepresentation]">
Sequence of message representations.

A message can be represented using the following formats:

1. `BaseMessagePromptTemplate`
2. `BaseMessage`
3. 2-tuple of `(message type, template)`; e.g.,
    `("human", "&#123;user_input&#125;")`
4. 2-tuple of `(message class, template)`
5. A string which is shorthand for `("human", template)`; e.g.,
    `"&#123;user_input&#125;"`
</ParamField>

<ParamField path="schema" type="dict | type">
A dictionary representation of function call, or a Pydantic model.
</ParamField>

<ParamField path="**kwargs" type="Any" default="&#123;&#125;">
Any additional kwargs to pass through to
`ChatModel.with_structured_output(schema, **kwargs)`.
</ParamField>

**Returns:** `ChatPromptTemplate`

A structured prompt template

**Examples:**

<CodeBlock showLineNumbers={false}>

```python
Instantiation from a list of message templates:

```python
from langchain_core.prompts import StructuredPrompt


class OutputSchema(BaseModel):
    name: str
    value: int


template = StructuredPrompt(
    [
        ("human", "Hello, how are you?"),
        ("ai", "I'm doing well, thanks!"),
        ("human", "That's good to hear."),
    ],
    OutputSchema,
)
```
