Create k-shot example selector using example list and embeddings.

Reshuffles examples dynamically based on Max Marginal Relevance.

**Parameters:**

<ParamField path="examples" type="list[dict]">
List of examples to use in the prompt.
</ParamField>

<ParamField path="embeddings" type="Embeddings">
An initialized embedding API interface, e.g. OpenAIEmbeddings().
</ParamField>

<ParamField path="vectorstore_cls" type="type[VectorStore]">
A vector store DB interface class, e.g. FAISS.
</ParamField>

<ParamField path="k" type="int" default="4">
Number of examples to select.
</ParamField>

<ParamField path="fetch_k" type="int" default="20">
Number of `Document` objects to fetch to pass to MMR algorithm.
</ParamField>

<ParamField path="input_keys" type="list[str] | None" default="None">
If provided, the search is based on the input variables
instead of all variables.
</ParamField>

<ParamField path="example_keys" type="list[str] | None" default="None">
If provided, keys to filter examples to.
</ParamField>

<ParamField path="vectorstore_kwargs" type="dict | None" default="None">
Extra arguments passed to similarity_search function
of the `VectorStore`.
</ParamField>

<ParamField path="vectorstore_cls_kwargs" type="Any" default="&#123;&#125;">
optional kwargs containing url for vector store
</ParamField>

**Returns:** `MaxMarginalRelevanceExampleSelector`

The ExampleSelector instantiated, backed by a vector store.
